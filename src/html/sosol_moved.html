The prototype version of the Son of SUDA Online editing environment for the Perseids Project and the Perseus Digital Library has moved.

The new location is http://sosol.perseids.org/sosol

User data from this prototype environment (i.e. the one at sosol.perseus.tufts.edu) has been archived but not carried over automatically to the new location. If you had work in progress that you would like ported to the new Perseids site please email bridget.almas@tufts.edu.
